JX Project Wizard: Changes from previous versions

Please submit all suggestions and bugs via github.

4.0

  • Removed support for code prefix.
  • Correctly displays all Unicode characters.
  • Window layouts adapt to translated text widths and font size differences.

3.0

  • Anti-alised text.

2.5

  • No longer copies version control directories when building a project from a template.

2.1

  • Projects can be placed anywhere, not merely in JX-2.1.0/programs/.
  • Changed file name convention to use _pre_ and _Binary_ instead of <pre> and <Binary>.
  • Automatically creates the project directory if it doesn't exist.
  • Added library template in lib_template.
  • Project path and name can now contain spaces.

2.0

  • Initial release
