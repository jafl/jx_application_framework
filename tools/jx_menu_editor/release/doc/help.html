Overview of JX Menu Editor

Overview summary

Main Window

Main window explanation.

To edit a text item, double click on it. To change the menu item type or icon, click in the cell to get a menu of the available options. To toggle the separator between items, click the cell.

The menu text and menu shortcut are displayed when the menu is opened. The menu text can be anything you want. The menu shortcut can include any combination of the following prefix modifiers:

  • Shift-
  • Ctrl-
  • Meta- (This key is labeled Alt on PC keyboards.)
  • Alt-
  • Super-
  • Hyper-
  • Mod1- through Mod5-
  • F1 through F35

To rearrange items, simply click and drag. Drag-and-Drop also works between windows. To move (rather than copy) an item from one window to another, hold down the Meta key while you drag the item.
