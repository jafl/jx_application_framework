Overview of JX Menu Editor

This program provides a graphical editor for defining menus to be displayed via JXTextMenu. It automatically generates a header file for inclusion in your code and a strings file allowing translation of both the text and the shortcuts.

The header file includes the menu definition string, the enum defining the menu item indexes, and a function that configures the menu item images. You should always call this configuration function after the menu items have been added to the menu.

Main Window

The class name defines the namespace for the item ID's. This is important for translation.

The menu title should be the text displayed in the menu bar. (This is not relevant for sub-menus and popup context menus, but it still helps explain the purpose of the menu.) The title is used to build the name of the variable containing the menu definition string passed to JXTextMenu::SetMenuItems, et. al., and the name of the function that you must call to configure the menu.

The Windows Alt shortcut is the key (combined with Meta) that will open the menu when the user has selected Windows-style menus.

Add items to the menu by using "New item" on the Table menu. You can rearrange items by dragging. Drag-and-Drop also works between windows, so you can move (hold down the Meta key) or copy items between menus.

The columns in the table are:

Type
This can be Plain (blank), Checkbox, or Radio Button.
Icon
The icon displayed for the menu item, which is also used by JXToolBar. The popup menu displays all the icons in the image-build directory inside your project directory and all the icons from the JX image directory (jx-af/image/jx).
Text (required)
The text of the menu item.
Key
The shortcut that will trigger the menu item when the menu is open. This will only be displayed when the user has selected Windows-style menus.
Shortcut
The shortcut that will trigger the menu item even if the menu is not open. You must define your shortcuts for the Windows-style menu. If the user selected Macintosh-style menus, the shortcuts will automatically be adjusted.
The menu shortcut can include any combination of the following prefix modifiers:
  • Shift-
  • Ctrl-
  • Meta- (This key is labeled Alt on PC keyboards.)
  • Alt-
  • Super-
  • Hyper-
  • Mod1- through Mod5-
  • F1 through F35
ID
The id of the menu item. If you do not specify a namespace with ::, the class name will be used. The ID is used as the key to retrieve translations and configure JXToolBar. (If you leave the cell blank, it will automatically be filled with a default starting with __. This indicates that JXToolBar should ignore the item.)
Enum Name (required)
The name of the enum constant defining the index of the menu item.
Separator
Click this cell to toggle displaying a separator after the menu item.

To edit a text item, double click on it. To change the menu item type or icon, click in the cell to get a menu of the available options. To toggle the separator between items, click the cell.

While editing, you can switch to a different item by using the following shortcuts:

Import Dialog

The import feature is provided to simplify upgrading existing applications. The dialog displays three fields:

Menu definition text (required)
Paste your existing code that defines the menu. This should be a text string enclosed in double quotes. It can also contain macros that expand to strings.
Menu items enum
Paste your existing code that defines the enum of the menu item indices.
actionDefs file
If you have a file that defines macros used in the menu definition, specify the full path to enable parsing of the menu definition.
