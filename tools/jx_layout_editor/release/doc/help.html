Overview of JX Layout Editor

Overview summary

Shift to turn off snap-to-grid

Create widget by meta-dragging.

Meta to toggle move/copy during Drag-and-Drop.

Selected widgets must all have same parent. Cannot add widget to selection if different parent.

Arrow keys to move selected widgets.

Meta-arrow keys to resize selected widgets.

Fixed edges marked orange.

Multiple shortcuts: concatenate without separator, e.g., ^C#.
