NPS Subversion® Client: Changes from previous versions

Please submit all suggestions and bugs via github.

2.0

  • Correctly displays all Unicode characters.

1.4

  • Fixed broken display of Subversion repository contents.
  • Correctly displays all Unicode characters.

1.3

  • Window layouts adapt to translated text widths and font size differences.

1.2

  • Added drag-and-drop on the Repo tab for tagging and branching.

1.1

  • Added Check Out items on File menu.
  • Added history menu in "Browse Repository" dialog, so you don't have to re-type frequently accessed URL's.
  • When you ask to browse a repository that is contained inside an existing window's repository, the sub-repository will be opened in the Repo tab.

1.0.3

  • Work around for strange X11 behavior that could cause Code Crusader to hang when run for the first time.

1.0.2

  • Anti-alised text.

1.0.1

  • Fixed crash when command line action is invoked while Subversion is busy.

1.0.0

  • Initial release
