Drakon Process Manager: Changes from previous versions

Please submit all suggestions and bugs via github.

2.0

  • Correctly displays all Unicode characters.

1.3

  • Window layouts adapt to translated text widths and font size differences.

1.2.5

  • Updated URL for version check.

1.2.4

  • Allow signaling other user's process via sudo.

1.2.3

  • Fixed display of processes that die while Drakon is updating the status.

1.2.2

  • Fixed crash when process disappeared while Drakon was updating the process' status.

1.2.1

  • Tweaked rendering of memory usage pie chart to make it look nicer.

1.2.0

  • Added display of CPU history and memory usage.

1.1.2

  • Preserves ownership of preferences file if you run as root.

1.1.1

  • Anti-aliased text.

1.1

  • Added incremental search to List view.
  • Full command line display field at the bottom of the window no longer has focus when Drakon starts.
  • Works better with virtual desktops. When invoked a second time, it jumps to the active desktop.
  • Mouse wheel now scrolls smaller increments on OS X.
  • Added support for horizontal scrolling. This is especially useful on Macintosh trackpads that support gestures.

1.0.0

  • Initial release
